```shell
gem install rails -v 6.0.2.1 

rails -v
```

```shell
cd ~/road-on-rails
rails _6.0.2.1_ new hello_app
```

### 名词解释
ruby: 一种编程语言，红宝石
gem, RubyGems: ruby 的包管理器
ruby on rails, rails: 用 ruby 编写的 web 应用框架 (A web-app framework) 
framework: 架构，框架，脚手架。比较抽象宽泛的概念，笔者暂时无法给出精确定义
